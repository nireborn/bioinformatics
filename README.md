# README

#### 1.如何运行：

##### 需要环境：

- python：＞＝v3.10

##### 运行方法：

在python编译器中运行主函数 `"main\GUI.py"` 。

#### 2.实现功能：

- 序列编辑
  - 序列剪切
  - GC含量统计
- fasta/fastq文件处理
  - 单碱基替换
  - 互补序列
  - 反向互补序列
  - 长度计算
- gff文件处理
- 可视化功能
  - 散点图
  - 折线图
  - 柱状图
  - 饼状图
  - 热力图
  - Venn图
  - 基因结构
  - 进化树

#### 3.测试数据：

测试数据均存放在`"data"`文件夹中。每一个功能对应一个数据文件，详见文件名。

> 例如：序列功能对应`01_seq.txt`文件。

数据格式需要按照测试数据格式存放，具体说明在平台对应功能的分页中。
