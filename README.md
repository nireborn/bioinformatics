# README

## 1.如何运行：

### 需要环境：

- python：＞＝v3.10

### 运行方法：

在python编译器中运行 `"main\GUI.py"` 。（由于打包成exe文件需要300MB，未开通会员，故无法上传。）

## 2.实现功能：

- 序列编辑
  - 序列剪切
  - GC含量统计
- fasta/fastq文件处理
  - 单碱基替换
  - 互补序列
  - 反向互补序列
  - 长度计算
- gff文件处理
- 可视化功能
  - 散点图
  - 折线图
  - 柱状图
  - 饼状图
  - 热力图
  - Venn图
  - 基因结构
  - 进化树

## 3.测试数据：

测试数据均存放在`"data"`文件夹中。

数据格式需要按照测试数据格式存放，具体说明在软件分页中说明。
